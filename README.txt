Code can be run through the console by typing

python cachesimulator.py <input_file.txt>

For example:
python cachesimulator.py input.txt
will run the cache simulator using the input file of input.txt

Assumes that the input file is in the same folder as cachesimulator.py
Dumps (cache.txt and ram.txt) will be created/updated in the same folder as cachesimulator.py
Assumes user inputs are both correct and within the menu options given in the project summary
Bonus is implemented (LFU replacement policy is functional)
